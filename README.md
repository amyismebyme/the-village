# The Village

The Village is a cloud-native community platform focused on helping men build meaningful real-world connections.

This project is intentionally designed as a production-grade engineering system to demonstrate Site Reliability Engineering (SRE), cloud infrastructure, DevOps, and software engineering best practices.

## Objectives

- Build a scalable cloud-native application
- Demonstrate Google SRE practices
- Implement observability from day one
- Deploy using Kubernetes
- Automate infrastructure using Terraform
- Practice incident response and reliability engineering

## Tech Stack

Frontend
- React
- Next.js
- TypeScript

Backend
- Go

Infrastructure
- Docker

Observability
- Prometheus
